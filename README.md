# Pokemon Tinder

A React + TypeScript + Vite app for swiping through Pokemon.
User can sign up and log in. Pokemon profiles will be displayed to user, including Pokemon name "Josh the Charmander", picture, stats and a random Chuck Norris Quote. User has the option to like/dislike each profile. Disliked Pokemon should not be shown again. Liked Pokemon are saved and can viewed later on. User can specify location area, gender and pokemon type.

## Setup

```bash
npm install
npm run dev
```

Copy and paste url after "Local: " into preffered browser.

## Build

```bash
npm run build
npm run preview
```

Copy and paste url after "Local: " into preffered browser.
